# Stack (Стек)

## Описание

Stack (Стек) - это структура данных LIFO (Last In, First Out), где последний добавленный элемент удаляется первым. Подобно стопке тарелок: вы кладете тарелки сверху и берете их также сверху.

## Характеристики

- **Принцип**: LIFO (Last In, First Out) - Последним вошел, первым вышел
- **Временная сложность**:
  - Push (добавление): O(1)
  - Pop (удаление): O(1)
  - Peek (просмотр): O(1)
  - IsEmpty/Size: O(1)
- **Пространственная сложность**: O(n), где n - количество элементов

## Использование

```go
package main

import (
    "fmt"
    "algos/stacks"
)

func main() {
    // Создаем новый стек
    s := stacks.New[int]()
    
    // Добавляем элементы
    s.Push(1)
    s.Push(2)
    s.Push(3)
    
    fmt.Println("Размер:", s.Size())  // 3
    fmt.Println("Стек:", s.String())  // [1 2 3]
    
    // Просматриваем верхний элемент
    top, _ := s.Peek()
    fmt.Println("Верхний элемент:", top)  // 3
    
    // Удаляем элементы
    item, _ := s.Pop()
    fmt.Println("Удалено:", item)  // 3
    
    // Проверяем пустоту
    if s.IsEmpty() {
        fmt.Println("Стек пуст")
    } else {
        fmt.Println("Стек не пуст")
    }
    
    // Очищаем стек
    s.Clear()
}
```

## Методы

### New[T any]() *Stack[T]
Создает новый пустой стек.

### Push(item T)
Добавляет элемент на вершину стека.
- Временная сложность: O(1)

### Pop() (T, error)
Удаляет и возвращает элемент с вершины стека.
- Возвращает ошибку `ErrStackEmpty`, если стек пуст
- Временная сложность: O(1)

### Peek() (T, error)
Возвращает элемент на вершине стека без его удаления.
- Возвращает ошибку `ErrStackEmpty`, если стек пуст
- Временная сложность: O(1)

### IsEmpty() bool
Возвращает `true`, если стек пуст, иначе `false`.

### Size() int
Возвращает количество элементов в стеке.

### Clear()
Удаляет все элементы из стека.

### Copy() *Stack[T]
Создает копию стека с теми же элементами.
- Временная сложность: O(n)

### ToSlice() []T
Возвращает представление стека в виде слайса.
- Первый элемент - это дно стека
- Последний элемент - это вершина стека

### String() string
Возвращает строковое представление стека.

## Применение

Стек широко используется в:

1. **Парсинг и компиляция**: проверка сбалансированности скобок, вычисление выражений
2. **Undo/Redo операции**: отмена действий в редакторах
3. **Backtracking**: поиск с возвратом
4. **Рекурсия**: использование стека вызовов функций
5. **Обратная польская нотация**: вычисление выражений
6. **Алгоритмы на графах**: DFS (поиск в глубину)
7. **Управление памятью**: управление стеком фреймов
